5.1 น้ำประปา

5.1.1 น้ำประปาที่ใช้มาจากแหล่งใด

1. น้ำประปานครหลวง ใช้ ไม่ใช้ กรุณาเลือก 1.2 มีปัญหาคุณภาพน้ำหรือไม่ ไม่มีปัญหา มีปัญหา คือ กรุณาเลือก ขุ่น/ตะกอน เค็ม/กร่อย กระด้าง มีกลิ่น คราบมัน ฝ้าขาว กรุณาเลือก 1.3 ปริมาณน้ำประปาที่ใช้เฉลี่ยเดือนละเท่าไหร่
ทราบปริมาณน้ำ ไม่ทราบปริมาณน้ำ แต่ทราบค่าน้ำ ไม่ทราบปริมาณน้ำ และค่าน้ำ เนื่องจากไม่มีมิเตอร์แยกจากส่วนกลาง ไม่ทราบปริมาณน้ำ และค่าน้ำ เนื่องจากได้รับการยกเว้น
ทราบปริมาณน้ำ (ลบ.ม./เดือน) กรุณากรอกจำนวน ทราบค่าน้ำ (บาท/เดือน) กรุณากรอกจำนวน
2. น้ำประปาส่วนภูมิภาค ใช้ ไม่ใช้ กรุณาเลือก 1.2 มีปัญหาคุณภาพน้ำหรือไม่ ไม่มีปัญหา มีปัญหา คือ กรุณาเลือก ขุ่น/ตะกอน เค็ม/กร่อย กระด้าง มีกลิ่น คราบมัน ฝ้าขาว Please select 1.3 ปริมาณน้ำประปาที่ใช้เฉลี่ยเดือนละเท่าไหร่

3. น้ำประปาอื่นๆ ใช้ ไม่ใช้ กรุณาเลือก 1.2 มีปัญหาคุณภาพน้ำหรือไม่ ไม่มีปัญหา มีปัญหา คือ กรุณาเลือก ขุ่น/ตะกอน เค็ม/กร่อย กระด้าง มีกลิ่น คราบมัน ฝ้าขาว Please select 1.3 ปริมาณน้ำประปาที่ใช้เฉลี่ยเดือนละเท่าไหร่
